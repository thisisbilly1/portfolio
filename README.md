## Tech used
- React
- React Bootstrap
- Font Awesome
- Three.js
- Nodejs Express
- Nodemailer

## Run back-end locally
Copy the `.env.example` and create your own `.env` file

Note that Gmail hates being used for automated emails.
```
npm i
npm start
```

## Run front-end locally
```
cd front
npm i
npm start
```
